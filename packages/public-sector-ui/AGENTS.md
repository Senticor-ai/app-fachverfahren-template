# Public Sector UI Instructions

Diese Regeln gelten zusätzlich zu `../../AGENTS.md`.

- UI-Komponenten bleiben domain-neutral und importieren keine Domain-Module.
- Sichtbare UI-Texte sind deutsch; Code-IDs und Exports bleiben englisch.
- Neue öffentliche Exports brauchen Storybook-Abdeckung und müssen die
  semantischen Design-Token-Aliasse nutzen.
