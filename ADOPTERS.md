# Adopters

Dieses Template ist für Wiederverwendung und Adaption gedacht. Projekte,
Organisationen und Verwaltungseinheiten können sich hier eintragen, wenn sie das
Template produktiv, pilotierend oder evaluierend nutzen.

| Organisation                   | Projekt oder Fachverfahren | Status | Kontakt oder Link |
| ------------------------------ | -------------------------- | ------ | ----------------- |
| Noch kein öffentlicher Eintrag | -                          | -      | -                 |
